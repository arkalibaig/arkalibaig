### Selected Projects

**[Karakoram-Trends](https://github.com/arkalibaig/Karakoram-Trends)**
35-year bias-corrected climate analysis of Gilgit-Baltistan using ERA5 reanalysis. Identified and corrected a known ECMWF 2016 data discontinuity that was inflating apparent warming by 16x across 5 stations. Dataset published on Kaggle.

**[digit_recognition_cnn](https://github.com/arkalibaig/digit_recognition_cnn)**
Handwritten digit classifier using a CNN trained on MNIST. Live Gradio interface for real-time predictions.

**[mobile-llm-inference](https://github.com/arkalibaig/mobile-llm-inference)**
Privacy-focused tool for accessing local LLM inference from a phone over Wi-Fi. Built with Ollama and Streamlit.
